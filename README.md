# Збий Шахед — Telegram бот

Цей комплект перетворює гру на Telegram Mini App: бот надсилає кнопку "Грати",
яка відкриває гру прямо в Telegram, а після завершення гри можна одним натиском
надіслати результат назад у чат.

## Що тут є

- `shahed-game.html` — сама гра, вже з підтримкою Telegram Web App SDK
  (адаптується під вʼюпорт Telegram, блокує випадкове закриття свайпом,
  має кнопку "Надіслати результат у чат"). Працює і як звичайна вебсторінка —
  можна відкрити просто в браузері для тесту.
- `bot.py` — бот на aiogram 3, який надсилає кнопку запуску гри та приймає
  результати.
- `requirements.txt` — залежності для бота.

## Крок 1. Створити бота

1. Напишіть [@BotFather](https://t.me/BotFather) в Telegram.
2. Команда `/newbot`, дайте назву та юзернейм.
3. BotFather видасть токен виду `123456789:AAExampleToken` — збережіть його.

## Крок 2. Розмістити гру на HTTPS

Telegram Web App відкриває **тільки HTTPS-посилання**, тому `shahed-game.html`
потрібно кудись завантажити. Файл повністю самодостатній (всі картинки вшиті
всередину), тож підійде будь-який статичний хостинг. Найпростіше —
**GitHub Pages**:

1. Створіть новий публічний репозиторій на GitHub.
2. Завантажте туди `shahed-game.html` (можна перейменувати на `index.html`).
3. Settings → Pages → Source: гілка `main`, папка `/ (root)` → Save.
4. Через хвилину гра буде доступна за адресою на кшталт
   `https://your-username.github.io/repo-name/`.

Альтернативи без реєстрації репозиторію: [Netlify Drop](https://app.netlify.com/drop)
(перетягнути файл у браузері) або Cloudflare Pages — обидва видають HTTPS-посилання
за секунди.

## Крок 3. Запустити бота

```bash
pip install -r requirements.txt

export BOT_TOKEN="ваш_токен_від_BotFather"
export WEBAPP_URL="https://your-username.github.io/repo-name/"

python bot.py
```

Бот працює через polling — тримайте процес запущеним (на сервері, VPS,
Raspberry Pi або через будь-який хостинг для Python-процесів).

## Крок 4. Перевірити

Знайдіть свого бота в Telegram, надішліть `/start` — з'явиться кнопка
"🎮 Грати". Після завершення гри кнопка "НАДІСЛАТИ РЕЗУЛЬТАТ У ЧАТ" надішле
рахунок назад у чат, і бот відповість повідомленням на кшталт:

> 🎯 Олег збив(ла) 24 дронів і дійшов(шла) до хвилі 4!

## Опціонально: кнопка меню

Щоб гра відкривалась не лише через `/start`, а й через постійну кнопку поруч
із полем вводу:

1. [@BotFather](https://t.me/BotFather) → `/mybots` → ваш бот → **Bot Settings**
   → **Menu Button** → **Configure Menu Button**.
2. Вкажіть той самий `WEBAPP_URL` та текст кнопки (наприклад "Грати").

## Нотатки

- Файл гри однаково добре працює і всередині Telegram, і як звичайна
  сторінка в браузері — Telegram-специфічний код вмикається лише якщо
  виявлено `window.Telegram.WebApp`.
- Якщо захочете оновити саму гру (баланс складності, новий паверап тощо),
  достатньо перезалити оновлений `shahed-game.html` на той самий хостинг —
  бот міняти не потрібно.
